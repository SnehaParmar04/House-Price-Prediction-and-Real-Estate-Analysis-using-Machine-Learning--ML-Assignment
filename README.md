# House Price Prediction

## Project Description
This project uses supervised machine learning models to predict house prices based on property features. The notebooks implement different experiments (RQ1–RQ7) including model comparison, preprocessing effects, feature importance, and robustness analysis.

---

## How to Run the Code

### On Kaggle (Recommended)
1. Upload the notebooks to Kaggle
2. Attach the dataset:
   https://www.kaggle.com/datasets/alyelbadry/house-pricing-dataset
3. Run all cells in each notebook

### On Local Machine
1. Clone the repository
2. Install required libraries:

pip install -r requirements.txt
3. Update dataset path if needed
4. Run notebooks using Jupyter
